<img width="1919" height="1079" alt="image" src="https://github.com/user-attachments/assets/9cefccf6-7259-477b-aaf0-aa457f81e234" />This project demonstrates a production-grade DevOps CI/CD pipeline designed to automate the build, test, and deployment of a web application using modern cloud and DevOps tools.
It simulates a real-world workflow by integrating Infrastructure as Code (IaC), configuration management, containerization, orchestration, and monitoring, ensuring scalable and reliable deployments.

The system follows a complete DevOps lifecycle:

GitHub – Source code management & webhook trigger
Jenkins – CI/CD pipeline automation
Terraform – Infrastructure provisioning (AWS)
Ansible – Configuration management (Jenkins setup)
Docker – Containerization of application
Kubernetes (EKS) – Container orchestration
Helm – Deployment management
SonarQube – Code quality analysis
Prometheus & Grafana – Monitoring & visualization
